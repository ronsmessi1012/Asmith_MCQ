# Study & Exam Platform

A comprehensive platform for employers to upload study materials and create exams, and for employees to study and take assessments with AI-powered feedback.

- **View Results**: specialized Excel-based tracking for all student scores.

## Architecture & Tech Stack

- **Frontend**: React (Vite), Tailwind CSS, Lucide Icons
- **Backend**: FastAPI (Python)
- **Database**: 
  - **Excel Integration**: Exams and results are stored in local Excel files for easy portability and access.
  - `exams_master.xlsx`: Registry of all exams.
  - `backend/exam_sheets/`: Individual Excel files for each exam's results.
- **AI Integration**: 
  - **Ollama (Llama 3)**: Used for generating exam questions and providing student feedback.
  - **PyPDF2**: For extracting text from study materials.

## Setup Instructions

### Prerequisites
- Python 3.9+
- Node.js 18+
- [Ollama](https://ollama.ai/) installed and running

### 1. Backend Setup

```bash
cd backend
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt

# Start Ollama Model (Required for Exams & Feedback)
ollama pull llama3:latest

# Run Server
uvicorn main:app --reload
```

The backend runs on `http://localhost:8000`.

### 2. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

The frontend runs on `http://localhost:5173`.

## File Structure

- `backend/`: FastAPI application
  - `main.py`: Core logic and API endpoints
  - `excel_utils.py`: Excel file handling (Master list + Per-exam sheets)
  - `exam_sheets/`: Storage for exam result Excel files
  - `uploaded_materials/`: Storage for PDF files
- `frontend/`: React application

## Usage

1. **Employer Login**: Click "Employer" and enter passcode (`admin123` by default).
2. **Upload**: Upload a PDF study guide.
3. **Create Exam**: Select the material, choose number of questions, and click "Create".
4. **Publish**: Review questions and publish.
5. **Employee View**: Click "Employee", enter name, and select an exam to take.
6. **Results**: Submit to see score and AI feedback. Check `backend/exam_sheets/` for the Excel record.
